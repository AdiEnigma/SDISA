# SDISA - Software Defined Image Spectrum Analyzer

SDISA is a Python-based image processing project that analyzes images in the frequency domain using the Fast Fourier Transform (FFT). The system converts an image into its frequency spectrum, applies frequency-domain filters, and reconstructs the processed image using Inverse FFT.

## Features

* Image preprocessing and normalization
* 2D FFT spectrum generation
* Spectrum visualization
* Low-pass, high-pass, and band-pass filtering
* Image reconstruction using IFFT
* Raspberry Pi deployment support

## Tech Stack

* Python
* NumPy
* OpenCV
* Matplotlib
* Git & GitHub
* Raspberry Pi

## Workflow

Image Input → Preprocessing → FFT → Spectrum Analysis → Frequency Filtering → IFFT → Reconstructed Image
